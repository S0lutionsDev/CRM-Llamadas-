# CRM LLAMADAS V1.1 - Instalador

Sistema de gestión de llamadas CRM para ventas y seguimiento de clientes.

## Descarga

Descarga el instalador desde la sección de [Releases](https://github.com/S0lutionsDev/CRM-Llamadas-/releases)

## Instalación

1. Descarga el archivo `CRM_LLAMADAS_V1.1_Setup_Final.exe`
2. Ejecuta el instalador
3. Sigue las instrucciones del asistente de instalación

## Requisitos del Sistema

- Windows 10 o superior
- 4 GB de RAM mínimo
- 500 MB de espacio en disco

Para soporte técnico, contacta al desarrollador.
